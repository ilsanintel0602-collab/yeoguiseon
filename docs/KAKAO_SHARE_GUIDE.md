## 📋 친구가 처음 사용할 때 안내

### 1단계: 링크 클릭

모바일 Chrome 또는 Safari에서 링크 열기:
```
https://ilsanintel0602-collab.github.io/yeoguiseon/app.html?v=24
```
(끝에 `?v=24`가 있으면 최신 버전 강제 로드)

### 2단계: 권한 허용

처음 열면 묻는 권한:
- **위치 권한** → "허용" (시군구 자동 설정)
- **카메라 권한** → "허용" (사진 분석)

### 3단계: 거주지 확인

화면 상단에 본인 시군구가 보이는지 확인:
- "일산동구", "강남구" 등 표시되면 OK
- "지역을 설정하세요"면 → 우측 상단 ⚙️ 설정 → 광역시/도 + 시/군/구 선택 + 저장

### 4단계: 사진 분석

화면 중앙 카메라 영역에서:
- 분리수거할 물건 찍기 (한 가지만, 가까이)
- 또는 갤러리에서 사진 선택
- 1~2초 뒤 결과 카드 표시

### 5단계: 피드백 (가장 중요!)

결과 카드 아래에 두 버튼:
- 👍 **도움 됐어요** (결과 맞음)
- 👎 **다른 거에요** (틀렸거나 헷갈림)

→ 클릭만 해도 우리에게 데이터로 들어와요!

## 🎯 친구한테 부탁할 핵심 5가지

피드백 받을 때 이 5개만 확인해도 충분:

| # | 사진 찍을 것 | 기대 결과 |
|---|---|---|
| 1 | 즉석밥 용기 | 플라스틱 (헹궈서) 또는 일반쓰레기 (오염 시) |
| 2 | 종이박스 (택배) | 종이류 (테이프 제거) |
| 3 | 노트북 / 가전 | 폐가전 무상수거 1599-0903 |
| 4 | 우유팩 / 멸균팩 | 종이팩 (행정복지센터 교환 가능) |
| 5 | 의류 (안 입는 옷) | 의류수거함 |

## 🐛 친구들이 만날 수 있는 문제 + 대응

### "사진 분석이 안 돼요"
- 와이파이/모바일 데이터 확인
- 다시 시도 (10초 정도 기다리기)

### "위치 자동 설정 안 됨"
- 모바일 Chrome 주소창 🔒 → 위치 권한 "허용"
- 또는 설정에서 직접 시군구 선택

### "결과가 이상해요"
- **그것 자체가 우리한테 가치 있는 피드백!**
- 👎 버튼 누르고 (선택) 어떤 결과 받았는지 카톡으로 알려주기

### "내 지역 정보가 없어요"
- 지금은 일산동구·덕양구·일산서구·파주·김포·의정부·부천 + 서울 강남·마포·은평·서대문 등 11개 시군구만 자세히
- 나머지는 환경부 표준 룰 (정확하지만 일반적)
- 친구 지역도 추가 필요하면 카톡으로 알려주기
