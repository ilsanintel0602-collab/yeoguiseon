## 자동 진행 흐름

```
[Phase N 시작]
   ↓
자산 점검 → 현 점수 측정
   ↓
작업 단계 1, 2, 3... (각 단계 후 검증)
   ↓
Phase 종결 자체 점검
   ├─ ≥ 95: ✅ 자동 Phase N+1 시작
   ├─ 90~94: ⚠️ 자동 보강 (최대 2회) → 재점검
   │           ├─ ≥ 95: ✅ Phase N+1
   │           └─ <95: 사용자 보고
   └─ <90: ❌ 사용자 결정 요청
```
