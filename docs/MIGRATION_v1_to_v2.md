# v1.1 → v2.0 마이그레이션 가이드

## 🎁 좋아진 점

- **API 키 없이 즉시 사용** — 진입 장벽 제거
- **오프라인 동작** — YOLO 모드는 인터넷 불필요
- **속도** — 두 번째 사용부터 LLM 호출 없이 즉시 결과
- **비용 0원** — 키 없이 무제한 사용

## ⚖️ 트레이드오프

- **첫 로드 5~15초** — 모델 다운로드 (1회만)
- **약 5MB 저장 공간** — 브라우저에 모델 캐시
- **YOLO 정확도 < LLM** — 일반 사물 충분, 미세 분류는 LLM 권장

## ⚠️ 기존 사용자가 알아야 할 점

1. **첫 실행 시 5~15초 로딩** — 깨진 게 아님, 모델 받는 중
2. **포트 변경** — v1=8001, v2=8002 (충돌 방지)
3. **기존 API 키 유지** — 자동으로 "정밀 분석" 옵션 활성화
4. **결과 카드 배지 추가** — 🤖 자동 인식 / ✨ 정밀 분석
5. **시크릿 모드** — 매번 모델 다운로드 (캐시 안 됨)

## 새 기능

- 🤖 자동 YOLO 인식 (API 키 없이)
- ✨ 정밀 LLM 분석 (API 키 있을 때 옵션)
- ⚠️ 낮은 신뢰도 경고
- 📡 GPS 자동 위치 (v1.1에 이미 있음)
- 🎯 분석 출처 배지 (YOLO/LLM 구분)

## v1과 v2 동시 실행 가능

- v1: 포트 8001 (yeoguiseon/start.bat)
- v2: 포트 8002 (yeoguiseon-v2/start.bat)
