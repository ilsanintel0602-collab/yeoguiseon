# 홈 화면에 여기선 설치하기 (PWA)

> 한 번 설치하면 카카오톡·인스타그램처럼 진짜 앱으로 동작해요.

## 왜 설치하나?

| 항목 | 브라우저 접속 | 홈 화면 설치 |
|---|---|---|
| 접속 시간 | 9~14초 (URL 입력) | **1~2초** (아이콘 탭) |
| 화면 | 주소창 보임 | **전체 화면** |
| 보이는가 | 즐겨찾기 깊숙이 | **홈 화면에 항상** |
| 잊혀짐 | 자주 | 거의 안 |

## Android (Chrome) 설치 방법

1. 브라우저에서 여기선 접속
2. 화면 하단에 **"홈 화면에 추가"** 배너 자동 등장
   - 또는: 메뉴(⋮) → "앱 설치" 또는 "홈 화면에 추가"
3. **"설치"** 또는 **"추가"** 클릭
4. ✅ 홈 화면에 ♻️ 여기선 아이콘 생성!

## iOS (Safari) 설치 방법

⚠️ iPhone은 **반드시 Safari** 사용 (Chrome iOS 아님)

1. Safari에서 여기선 접속
2. 하단 **공유 버튼** (네모+화살표 ⬆️) 누름
3. 메뉴에서 **"홈 화면에 추가"** 선택
4. 우상단 **"추가"** 누름
5. ✅ 홈 화면에 ♻️ 여기선 아이콘 생성!

## 설치 후 사용

홈 화면의 ♻️ 여기선 아이콘 누르면:
- 즉시 풀스크린 앱 화면
- 주소창·브라우저 메뉴 없음
- 카카오톡 같은 진짜 앱처럼 동작
- 오프라인에서도 기본 동작 (한 번 사용 후)

## 제거하고 싶을 때

다른 앱처럼:
- **Android**: 아이콘 길게 누름 → "삭제" 또는 "제거"
- **iOS**: 아이콘 길게 누름 → "앱 삭제"

## 자주 묻는 질문

### Q: 설치 배너가 안 떠요
- **Android**: HTTPS 환경 필요 (localhost 또는 Vercel 배포 시)
- **iOS**: Safari에서 수동으로 공유 → "홈 화면에 추가"
- 한 번 거부하면 24시간 후 다시 표시

### Q: 앱스토어에서 검색되나요?
- ❌ PWA는 앱스토어에 없어요. URL로만 공유 가능
- 향후 정식 앱 출시 시 앱스토어 등록 예정

### Q: 알림은 어떻게 받나요?
- 향후 추가 예정 (배출일 알림 등)
- 현재는 알림 기능 미지원

### Q: 데이터는 안전한가요?
- ✅ 모든 데이터는 본인 폰 브라우저에만 저장
- ✅ API 키도 본인 폰에만 (외부 전송 X)
- ✅ 촬영한 사진은 AI 분석 후 즉시 폐기 (저장 안 됨)

## 기술 정보

- 모델 다운로드: 약 5MB (최초 1회)
- 캐시 사용: 약 10MB (앱 셸 + AI 모델)
- 오프라인 가능: ✅ (모델 다운로드 후)
- 푸시 알림: 향후 지원 예정
