## 작업 표준 (각 영역 동일 패턴)

```
1. publicDataPk 확정 (data.go.kr 검색)
2. crawl_<영역>.py 작성 (crawl_bunribaechul.py 패턴 재사용)
3. data/<영역>_raw.json 저장 (원본 보존)
4. data/<영역>.json 변환 (우리 스키마)
5. app.html 통합:
   - loadAllData() fetch 추가
   - 전역 변수 선언 (예: let MEDICINE_BINS = null;)
   - renderResult 카드에 활용 코드 추가
6. DATA_INVENTORY.md 갱신
7. data_audit_full.py 자동 검증
8. WORK_HISTORY.md 작업 기록
```

→ data-steward SKILL.md의 SOP 11단계 그대로 적용.

- 행안부 공공데이터포털 검색: "폐의약품"
- 시군구별 보건소·약국 목록
- 행안부 #15012005 또는 동등 데이터셋

### 통합 위치
`app.html` line 1466 근처 (medicine 카테고리 분기에)

### 작업 산출물
- `scripts/crawl_medicine_bins.py`
- `data/medicine_bins.json` (전국 약국·보건소 폐의약품 수거함)
