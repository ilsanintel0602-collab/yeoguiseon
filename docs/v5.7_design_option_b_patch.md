## 패치 코드

### 1. `detectionState` — 라이브 detect 비활성화 토글 추가

기존 (line 1130 근처, `state` 또는 비슷한 곳):
```js
const detectionState = {
  active: false,
  rafId: null,
  detections: [],
  lastDetectTime: 0,
};
```

→ 그대로 두고, **`startRealtimeDetection` 함수 안의 `detect` 호출만 비활성화**.

### 2. `startRealtimeDetection` — detect 호출 제거, 가이드만 그리기

기존 (line 1146~1176):
```js
const detectLoop = async () => {
  if (!detectionState.active) return;
  const now = Date.now();
  if (now - detectionState.lastDetectTime > 500 && video.videoWidth > 0 && yoloModel) {
    // ... yoloModel.detect 호출 ...
  }
  drawBoxes(canvas);
  detectionState.rafId = requestAnimationFrame(detectLoop);
};
```

→ 변경:
```js
// v5.7: 라이브 detect 제거 + 우아한 촬영 가이드만 그리기
const detectLoop = (timestamp) => {
  if (!detectionState.active) return;
  drawCameraGuide(canvas, timestamp);  // ← detect 없음, 가이드만
  detectionState.rafId = requestAnimationFrame(detectLoop);
};
```

`yoloModel.detect` 호출 통째 제거. `lastDetectTime`, `detections` 배열도 비활성화 (호환성 위해 코드는 유지하되 빈 배열).

### 3. `drawCameraGuide` — 새 함수 (기존 `drawBoxes` 대체)

`drawBoxes` 함수 (line 1188) 옆에 추가:

```js
// v5.7 디자인 옵션 B: 우아한 촬영 가이드
function drawCameraGuide(canvas, timestamp = 0) {
  const ctx = canvas.getContext('2d');
  const w = canvas.width;
  const h = canvas.height;
  ctx.clearRect(0, 0, w, h);

  // 황금비율 영역 (중앙 80%)
  const padX = w * 0.1;
  const padY = h * 0.15;
  const innerW = w - padX * 2;
  const innerH = h - padY * 2;

  // 4개 모서리 brackets (각 모서리에서 안쪽으로 8% 길이)
  const armX = innerW * 0.08;
  const armY = innerH * 0.08;

  // 펄스 효과 (1.5초 주기, 0.6~1.0 알파)
  const pulse = 0.6 + 0.4 * Math.sin(timestamp / 750);

  ctx.strokeStyle = `rgba(0, 255, 136, ${pulse})`;
  ctx.lineWidth = 4;
  ctx.lineCap = 'round';
  ctx.shadowBlur = 16;
  ctx.shadowColor = '#00ff88';

  // 좌상단
  ctx.beginPath();
  ctx.moveTo(padX, padY + armY);
  ctx.lineTo(padX, padY);
  ctx.lineTo(padX + armX, padY);
  ctx.stroke();

  // 우상단
  ctx.beginPath();
  ctx.moveTo(w - padX - armX, padY);
  ctx.lineTo(w - padX, padY);
  ctx.lineTo(w - padX, padY + armY);
  ctx.stroke();

  // 좌하단
  ctx.beginPath();
  ctx.moveTo(padX, h - padY - armY);
  ctx.lineTo(padX, h - padY);
  ctx.lineTo(padX + armX, h - padY);
  ctx.stroke();

  // 우하단
  ctx.beginPath();
  ctx.moveTo(w - padX - armX, h - padY);
  ctx.lineTo(w - padX, h - padY);
  ctx.lineTo(w - padX, h - padY - armY);
  ctx.stroke();

  // 중앙 십자선 (작은 + 표시, 펄스)
  const cx = w / 2;
  const cy = h / 2;
  const crossLen = 12;
  ctx.shadowBlur = 8;
  ctx.lineWidth = 2;
  ctx.beginPath();
  ctx.moveTo(cx - crossLen, cy);
  ctx.lineTo(cx + crossLen, cy);
  ctx.moveTo(cx, cy - crossLen);
  ctx.lineTo(cx, cy + crossLen);
  ctx.stroke();

  // 안내 텍스트 (선택, 하단)
  ctx.shadowBlur = 0;
  ctx.fillStyle = `rgba(255, 255, 255, ${0.7 * pulse})`;
  ctx.font = '14px -apple-system, "Apple SD Gothic Neo", sans-serif';
  ctx.textAlign = 'center';
  ctx.fillText('가이드 안에 사물을 두고 촬영하세요', cx, h - padY * 0.5);
}
```

### 4. `stopRealtimeDetection` — 그대로

```js
function stopRealtimeDetection() {
  detectionState.active = false;
  if (detectionState.rafId) cancelAnimationFrame(detectionState.rafId);
  const canvas = $('bboxOverlay');
  if (canvas) canvas.getContext('2d').clearRect(0, 0, canvas.width, canvas.height);
  const status = $('detectStatus');
  if (status) status.style.display = 'none';
}
```

변경 없음.

### 5. 라이브 박스 라벨 제거

`drawBoxes` 함수 (line 1188)는 사용 안 함. 또는 그대로 두고 호출만 안 함. 깔끔히 하려면 함수 통째 삭제.

### 6. COCO-SSD 모델 로드 — 조건부 (선택)

`loadYoloModel` 함수도 사실 라이브에서는 안 쓰는데, 촬영 후 폴백용으로 필요할 수도. 그대로 두기.

→ 사용자가 촬영 버튼 누르면 그때 yoloModel.detect (분석용) 호출. 라이브에서는 0.
