### 🛡️ 사용자 안전 보장 (app.html v5.25 line 489~)

**이미 안전망 추가됨.** matchRule이 region_exceptions name과 regions_meta name을 비교해서 명백히 다르면 **무시하고 환경부 표준만 사용**. 즉 잘못된 매핑이라도 사용자에게 표시되지 않음.

```javascript
// v5.25 안전망
if (REGIONS?.level2?.[currentRegion]?.name) {
  const metaCore = (metaName.match(/(\S+?)(시|구|군)$/) || [])[1];
  if (metaCore && metaCore.length >= 2 && !exName.includes(metaCore)) {
    break;  // 잘못 매핑 — 환경부 표준만 사용
  }
}
```
