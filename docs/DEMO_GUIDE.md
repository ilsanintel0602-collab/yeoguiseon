## Q&A 예상 답변

**Q1. 카메라 인식 정확도는?**
A: Gemini 3.5 Flash 기반, 한국 사물 패턴 SYSTEM_PROMPT로 정확도 강화. 박스 인식 1회당 3~5초, 분석 응답 5~7초. 다재질 물건(텀블러·생수병)은 자동으로 노란 박스로 "재질 확인 필요" 안내.

**Q2. 데이터는 얼마나 자주 갱신?**
A: 시·구청 룰 변경 발견 시 즉시 업데이트. Worker stale-while-revalidate 캐시로 사용자는 항상 최신 데이터. 봉투 가격은 행안부 API 동기화 가능.

**Q3. 비용 모델?**
A: Cloudflare Worker 무료 한도 + Gemini API 일일 1000회 제한. 시연·일반 사용 무료. 대규모 확장 시 비용 발생.

**Q4. 개인정보 처리?**
A: 사진은 Worker에 SHA-256 해시로 24시간 임시 캐시 (동일 사진 재분석 시 응답 가속). GPS는 즉시 사용 후 폐기. 이름·전화번호·계정 일체 수집 X.

**Q5. 다른 비슷한 앱과 차별점?**
A: 환경부 표준 100% 정합 + 출처 표시. 매장 정책·환급 안내 X (사실만 전달). 지역별 정확 안내 (261/261).

**Q6. 향후 계획?**
A: 다국어 (영문) 지원, MediaPipe 자체 모델 (오프라인 분석), 사용자 피드백 학습 시스템.

**Q7. 카메라 외 다른 인식 방법?**
A: 이름 검색·음성 검색 모두 즉시. 시각장애인·노인 친화적. 카메라 권한 거부해도 모든 기능 사용 가능.
