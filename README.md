# ♻️ 여기선 v3.0 — 대한민국 어디서나

> **전국 226개 시군구 어디서나 사용 가능한 분리수거 안내 앱**

## 핵심 가치

- 🇰🇷 **대한민국 어디서나** — 전국 시군구 100% 커버
- 🆓 **API 키 없이 시작** (브라우저 AI 내장)
- 📡 **GPS 자동 위치 인식**
- 🏛 **환경부 표준 + 지역별 차이** 모두 반영
- 👴 **60대 친화 UI** (큰 글씨, 큰 버튼)
- 📱 **iOS+Android+PC** (PWA)

## 빠른 시작 (30초)

1. **`start.bat` 더블클릭**
2. 브라우저: `http://localhost:8003/`
3. 첫 화면: "🤖 AI 준비 중..." (5~15초, 와이파이 권장)
4. ⚙️ → **거주지 광역시/도 + 시군구** 선택 (또는 GPS 자동)
5. 큰 버튼 → 사진 → 결과!

## v2 → v3 변화 (게임 체인저)

| 항목 | v2.2 | **v3.0** |
|---|---|---|
| 지원 지역 | 강남, 일산 (2개) | **전국 226개 시군구** |
| 사용 가능 인구 | 약 100만 | **5,200만** |
| 데이터 구조 | 지역별 룰만 | **환경부 표준 + 지역 예외** |
| GPS 매핑 | 강남/일산만 | **전국 17개 광역** |
| 모르는 지역 | 사용 불가 | **환경부 표준 적용** |

## 어떻게 동작?

```
사용자 위치 (GPS or 시도→시군구 선택)
   ↓
사진 → YOLO 인식 (브라우저 AI)
   ↓
사진 무엇? + 어느 지역?
   ↓
1) 지역 예외 룰 있나? → 적용
2) 없으면 → 환경부 표준 룰
   ↓
결과 표시 + 출처 배지
  - 📍 [지역] 전용 (예외 룰)
  - 🇰🇷 환경부 표준 (전국 공통)
```
