## 🎯 1. 기획안 (Vision)

### 한 줄 비전
> **"대한민국에서 가장 빠르고 정확한 분리수거 안내 앱 — 카메라 한 번이면 끝."**

### 핵심 가치
| 가치 | 의미 |
|---|---|
| 빠름 | 사진 찰칵 → 0.5~1.5초 안에 결과 |
| 정확 | 한국 정부 공식 데이터로 학습된 모델 (92%+) |
| 무료 | 사용자 비용 영구 0, 광고 0 |
| 오프라인 | 네트워크 없어도 작동 (분리수거장은 보통 지하·실외) |
| 최신 | 정부 정책 변경 자동 반영 (주 1회 자동 크롤링) |
| 투명 | 모든 안내에 출처·확인일 표시 |
| 프라이버시 | 사진 서버로 안 감, 기기 내 처리 |

### 사용자 (Target)
- 한국 거주자 누구나
- 특히: 분리수거 헷갈리는 까다로운 케이스가 자주 있는 사용자
- 디바이스: 모바일(메인) + PC

### 사용 시나리오
```
1. 사용자가 분리수거장에 쓰레기 들고 옴
2. 앱 열고 카메라로 찰칵
3. 0.5~1.5초 후 결과:
   "📦 종이박스
    카테고리: 종이류
    배출 방법: 1) 테이프 제거 2) 납작하게 펴기 3) 끈으로 묶기
    📍 일산동구 기준 — 새벽 4시 전까지 문 앞
    출처: 고양시청 (A등급) · 확인일: 2026-05-18"
4. 끝.
```

## 🏗️ 3. 시스템 구조

### 3.1 사용자 측 (PWA, 브라우저)
```
┌──────────────────────────────────────┐
│   카메라 / 사진 입력                  │
└──────────────────────────────────────┘
              ↓
┌──────────────────────────────────────┐
│   Layer 1: YOLO (한국 분리수거 특화) │ ← 메인
│           [AI Hub 학습, ~40MB]        │
└──────────────────────────────────────┘
       ↓ 신뢰도 낮으면 ↓
┌──────────────────────────────────────┐
│   Layer 2: 후보 목록 사용자 선택     │ ← 보조
└──────────────────────────────────────┘
       ↓ 그래도 모르면 ↓
┌──────────────────────────────────────┐
│   Layer 3: 수동 카테고리 선택         │ ← 마지막 보조
└──────────────────────────────────────┘
              ↓
┌──────────────────────────────────────┐
│   룰 매칭 (JSON DB)                   │
│   + 지역 자동 인식 (GPS)              │
└──────────────────────────────────────┘
              ↓
┌──────────────────────────────────────┐
│   결과 표시                           │
│   - 카테고리 / 배출 방법              │
│   - 우리 지역 특수 룰                 │
│   - 종량제봉투 가격                   │
│   - 출처·확인일                       │
└──────────────────────────────────────┘
```

### 3.2 백엔드 (없음, GitHub Pages만)
```
┌──────────────────────────────────────┐
│   GitHub 저장소                       │
│   - app.html (PWA)                   │
│   - data/*.json (룰 + 지역 + 봉투)   │
│   - models/yolo-recycle.tflite (40MB)│
└──────────────────────────────────────┘
              ↓ 자동 배포
┌──────────────────────────────────────┐
│   GitHub Pages (무료 호스팅)          │
└──────────────────────────────────────┘
              ↑ 주 1회 자동 갱신
┌──────────────────────────────────────┐
│   GitHub Actions (무료 크롤링)        │
│   - 환경부 분리배출.kr 크롤링         │
│   - 행안부 표준데이터 다운로드        │
│   - 변경 감지 → 자동 커밋             │
└──────────────────────────────────────┘
```

### 3.3 학습 (1회성, Colab)
```
AI Hub 신청 → 승인 (1-3일)
   ↓
570GB 데이터 다운 (Colab)
   ↓
YOLO v8 Small 학습 (3-5일)
   ↓
TF.js 변환 (~40MB)
   ↓
GitHub Release 업로드
   ↓
570GB 데이터 삭제 ✅
```

## 🎯 6. 성공 지표

### 기술 지표
- [ ] 카메라 정확도 ≥ 88% (50장 테스트 기준)
- [ ] 사진→결과 응답 시간 ≤ 1.5초 (중급 모바일 기준)

- [ ] 오프라인 작동 확인
