## 💎 오늘 학습 (메모리 박힘)

1. **AI Hub 해외 IP 차단**: Colab 같은 미국 IP에서 다운 불가. 대안 = 사용자 PC + Drive.
2. **MAX_TOKENS 함정**: maxOutputTokens 512 → 종이박스 OCR로 빠지면 응답 잘림. 1024가 안전.
3. **데이터 오염은 카테고리 단위**: 한 alias가 여러 item에 복붙되면 전체 정확도 망가짐. 카테고리/대형폐기물 같은 단어는 alias 부적합.
4. **자동 push의 가치**: Web Upload 비효율 + 위치 실수 위험. PAT + Git Data API가 정답.
5. **벤치마크가 본질 측정 도구**: 모바일 검증 X. 객관 점수 매 cycle 추적이 회귀 방지.
6. **PowerShell + CMD 한글 인코딩**: .bat은 ASCII만, .ps1은 `[Console]::OutputEncoding = UTF8`.
