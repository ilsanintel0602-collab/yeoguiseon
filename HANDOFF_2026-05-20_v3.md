1. https://dash.cloudflare.com → Workers & Pages → **yeoguiseon-proxy** → Edit code
2. `E:\Cowork 작업\yeoguiseon-v4\scripts\cloudflare_worker.js` 전체 내용 복사

## ⚠️ 알려진 주의사항

- **reusable_cup 매핑**: SYSTEM_PROMPT 예시 목록엔 있지만 `NATIONAL.items`에는 직접 항목 없을 수 있음. Gemini가 돌아오면 `category_hint='reusable'` 폴백으로 처리. COCO-SSD 단독일 때만 DB miss → 결과 화면이 raw key를 보일 수 있음. 모바일 테스트에서 확인 필요.
- **buildAllowList**: Worker가 새 Origin (https://ilsanintel0602-collab.github.io) 이미 포함. CORS는 문제 없음.
